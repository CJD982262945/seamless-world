## 目录结构

这里罗列下server、zeus的主要目录

## server目录

```tree
├─bin
├─doc
├─res
│  ├─config
│  ├─entitydef
│  ├─excel
│  └─space
├─sql
└─src
```

目录            | 说明
----------------|----------------------------------
bin             | 执行TimeFire/server/build.bat后，可执行文件目录
doc             | 文档目录
res/config      | 可执行文件基本配置目录
res/entitydef   | 实体定义目录
res/excel       | 业务逻辑配置目录
res/space       | 场景资源目录
sql             | mysql sql脚本
src             | 代码目录，详见 [src/ReadMe.md](../../src/ReadMe.md)
