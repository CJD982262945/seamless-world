## <http://%s:%s/login>

-   目前只支持Post
-   需要携带Post数据：

    ```json
    {"User":"YourAccount","Password":"YourPassword"}
    ```

-   返回JSON数据

    类似如下信息：
    ```json
    {
      "UID":6,
      "Token":"0cf7d072d17752d35fb6826b516cb026",
      "LobbyAddr":"localhost:2402",
      "Result":0,
      "ResultMsg":"登录成功",
      "HB":false
    }
    ```

-   根据`LobbyAddr`知道Gateway地址
