## EnterSpace

```go
type EnterSpace struct {
	SpaceID   uint64
	MapName   string
	EntityID  uint64
	Addr      string
	TimeStamp uint32
}
```

  - struct协议
  - room -> gateway -> client
  - 重要信息：SpaceID
  - 重要信息：Addr
  - 客户端可以连接Room服务器

## SpaceUserConnect
