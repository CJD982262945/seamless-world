# 配置上的一些坑

## 一、res/config/server.json

#### OuterAddr

要对外提供服务，需要设置为本机器IP

#### MySQLAddr

连接正确的Mysql

#### 4个 Redis 服务器地址 Addr

可以同一个redis不同的db, 更改 Index 为4个不同的值

#### Config.MaxLoad

机器差，可以把 Config.MaxLoad 改为 99

不然可能因CPU或内存占用高而出现登录时“服务器忙”（这个是百分比值）


#### GM账号

把自己修改成gm , 注意修改后换新账号，跳伞后客户端会有gm按钮

```json
"Login": {
	       "InitGrade": "0",\\此处修改为0，就是gm
	       ...
	     }
```
